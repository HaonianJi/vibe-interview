# Pomodoro Dashboard

Build a Pomodoro productivity application with a configurable timer, task association, session history log, and daily/weekly statistics dashboard.

## Requirements

### Core Features (Must Have)
1. **Pomodoro Timer**: A countdown timer defaulting to 25 min work / 5 min break, with start, pause, reset controls and configurable durations
2. **Task Association**: Maintain a task list and associate each Pomodoro session with a specific task so time spent per task is tracked
3. **Session History**: Log every completed Pomodoro with timestamp, duration, and associated task — display as a scrollable list grouped by date
4. **Productivity Stats**: A dashboard showing daily and weekly totals — number of Pomodoros completed, total focus time, and a bar chart of sessions per day for the past 7 days
5. **Notifications**: Play an audio alert and/or show a browser notification when a work or break period ends

### UI Requirements
- A prominent circular or linear timer display as the centerpiece of the app
- A task sidebar or dropdown for selecting the current task before starting a session
- A stats section with at least one chart (bar, line, or similar) for the weekly overview
- Clean transitions between work and break states with distinct visual themes for each

## Technical Notes
- You may use any tech stack
- You may use any libraries
- Focus on functionality first, then polish
