# 文件浏览器

构建一个虚拟文件浏览器应用，包含树形视图侧边栏、面包屑导航，以及创建、重命名、删除和预览文件与文件夹的功能。

## 需求

### 核心功能（必须实现）
1. **树形视图**：可折叠的树形侧边栏，展示完整的文件夹层级结构，带有展开/折叠切换和文件/文件夹图标
2. **文件与文件夹操作**：创建新文件和文件夹、重命名已有项目、删除项目时弹出确认提示
3. **面包屑导航**：顶部面包屑栏反映当前路径，每个路径段可点击以向上导航
4. **内容预览**：点击文件时显示预览面板——渲染纯文本内容，对图片或其他类型显示占位预览
5. **拖拽移动**：通过拖拽将文件和文件夹移动到树形视图或主内容区域中的其他文件夹

### 界面要求
- 双栏布局，左侧为树形侧边栏，右侧为内容/预览区域
- 拖拽悬停目标有视觉反馈
- 提供右键菜单或操作按钮用于创建、重命名和删除操作
- 响应式行为，在小屏幕上折叠侧边栏

## 技术说明
- 可使用任意技术栈
- 可使用任意库
- 优先实现功能，再进行优化打磨
