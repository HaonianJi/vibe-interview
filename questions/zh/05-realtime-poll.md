# 实时投票应用

构建一个支持结果实时更新的投票应用。

## 需求

### 核心功能（必须实现）
1. **创建投票**：提供表单以创建包含问题和 2-5 个选项的投票
2. **参与投票**：用户可选择一个选项并提交投票
3. **实时结果**：投票后，以柱状图或进度条的形式展示带百分比的投票结果
4. **投票列表**：首页展示所有进行中的投票
5. **数据持久化**：投票和投票结果在页面刷新后必须保留

### 投票规则
- 每个浏览器会话对每个投票只能投一次（使用 localStorage 或 cookies）
- 提交后不可更改投票
- 显示总投票数

### 界面要求
- 结果条带动画效果，新投票到来时自动更新
- 投票后提供清晰的视觉反馈
- 适配移动端的布局

## 技术说明
- 可使用任意技术栈
- "实时"可通过轮询（setInterval）、SSE 或 WebSocket 实现
- 注重流畅的投票和结果展示体验
