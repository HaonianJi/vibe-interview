# 电影观影清单

构建一个电影观影清单应用，用户可以通过免费 API 搜索电影、添加到个人观影清单、撰写评分和影评，并按类型或评分筛选。

## 需求

### 核心功能（必须实现）
1. **电影搜索**：通过免费 API（如 OMDB、TMDB）按片名搜索电影，展示搜索结果包含海报、片名、年份和类型
2. **观影清单管理**：将搜索结果中的电影添加到个人观影清单，标记电影为已看或未看，从清单中移除电影
3. **评分与影评**：为观影清单中的任何电影打星级评分（1-5 星）并撰写简短文字影评——在电影详情页展示评分和影评
4. **筛选与排序**：按类型、已看/未看状态或最低评分筛选观影清单，支持按片名、年份或用户评分排序
5. **电影详情页**：详情页展示来自 API 的完整电影信息（剧情简介、导演、演员、时长），以及用户的评分和影评

### 界面要求
- 搜索栏支持自动触发或按钮触发搜索，结果以网格或列表形式的电影卡片展示
- 观影清单页面在侧边栏或工具栏中提供筛选控件
- 星级评分输入组件用于电影评分
- 搜索结果和观影清单均采用响应式卡片布局

## 技术说明
- 可使用任意技术栈
- 可使用任意库
- 可使用任意免费电影 API（OMDB、TMDB 等）——请注明如何获取 API Key
- 优先实现功能，再进行优化打磨
