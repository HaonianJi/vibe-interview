# Markdown 笔记应用

构建一个支持 Markdown 语法的笔记应用。

## 需求

### 核心功能（必须实现）
1. **笔记增删改查**：创建、查看、更新和删除笔记
2. **Markdown 编辑器**：支持 Markdown 语法的文本编辑区域
3. **实时预览**：并排显示或切换显示渲染后的 Markdown 内容
4. **笔记列表**：侧边栏列出所有笔记，按最后修改时间排序
5. **搜索功能**：按标题和内容搜索笔记
6. **数据持久化**：所有笔记在页面刷新后必须保留

### Markdown 支持
至少需要渲染以下语法：标题、粗体、斜体、列表、代码块、链接

### 界面要求
- 分栏布局（笔记列表 | 编辑器 | 预览）或等效布局
- 当前选中的笔记在侧边栏中高亮显示
- 自动保存或提供显式保存按钮

## 技术说明
- 可使用任意技术栈
- 可使用 Markdown 渲染库（marked、react-markdown 等）
- 注重流畅的编辑体验
