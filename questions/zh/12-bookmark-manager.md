# 书签管理器

构建一个书签管理应用，用户可以保存、标记和整理书签到文件夹中，搜索所有已保存的链接，并导入或导出书签集合。

## 需求

### 核心功能（必须实现）
1. **保存书签**：添加书签时输入 URL、标题、可选描述和一个或多个标签——如果仅提供 URL，则自动获取页面标题
2. **文件夹管理**：创建嵌套文件夹结构，通过拖拽或移动对话框在文件夹之间移动书签
3. **标签与搜索**：为书签分配多个标签，提供搜索栏按标题、URL、描述或标签实时筛选
4. **导入/导出**：从 JSON 或 HTML 文件（浏览器书签导出格式）导入书签，并以相同格式导出当前书签集合
5. **批量操作**：选择多个书签执行批量操作——删除、移动到文件夹或添加/移除标签

### 界面要求
- 侧边栏展示文件夹树和标签云或标签列表，用于快速筛选
- 主区域以卡片或列表行形式展示书签，包含标题、URL 摘要和标签
- 提供弹窗或内联表单用于添加和编辑书签
- 视觉标识当前所在文件夹和已应用的标签筛选

## 技术说明
- 可使用任意技术栈
- 可使用任意库
- 优先实现功能，再进行优化打磨
