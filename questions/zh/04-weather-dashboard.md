# 天气仪表盘

构建一个展示当前天气状况和天气预报的仪表盘。

## 需求

### 核心功能（必须实现）
1. **城市搜索**：搜索城市并显示其当前天气
2. **当前天气**：展示温度、湿度、风速、天气描述以及相应的天气图标
3. **5 天预报**：展示未来 5 天的最高/最低温度和天气状况
4. **最近搜索**：展示最近搜索的 5 个城市以便快速访问
5. **单位切换**：在摄氏度和华氏度之间切换

### 数据来源
使用免费天气 API，可选方案：
- OpenWeatherMap（免费套餐）：https://openweathermap.org/api
- wttr.in（无需 API Key）：https://wttr.in/:help
- Open-Meteo（无需 API Key）：https://open-meteo.com/

### 界面要求
- 简洁的布局，数据层级清晰
- 数据加载时显示加载状态
- 对无效城市或 API 错误进行友好的错误处理

## 技术说明
- 可使用任意技术栈
- 必须调用真实 API（不可使用模拟数据）
- 优雅地处理 API 错误
