# 短链接服务

构建一个短链接服务，自动使用开放元数据 API 获取并展示每个短链接的富预览信息（标题、描述、缩略图）。

## 需求

### 核心功能（必须实现）
1. **URL 缩短**：接受长 URL 并生成唯一的短码——展示生成的短链接并提供一键复制按钮
2. **富链接预览**：URL 缩短时，自动使用免费 API 或服务端抓取获取其元数据（页面标题、描述、Open Graph 图片）——在短链接旁展示预览卡片
3. **自定义别名**：允许用户指定自定义别名替代随机短码，校验防止重复
4. **点击分析**：为每个短链接展示总点击次数，以及过去 7 天点击量的时序图表
5. **链接管理仪表盘**：表格列出所有短链接，包含预览缩略图、原始 URL、点击次数和创建日期

### API 集成
- 使用**免费元数据/预览 API**（如 jsonlink.io、microlink.io 或 Open Graph 抓取）获取每个 URL 的页面标题、描述和缩略图
- 展示包含获取到的元数据（图片、标题、描述、域名）的富预览卡片
- 异步获取和更新预览——获取期间显示加载骨架屏
- 优雅处理没有 Open Graph 数据的 URL（使用域名 + favicon 作为回退方案）
- 缓存预览数据以避免重复 API 调用

### 界面要求
- 简洁的落地页，突出显示 URL 输入框和缩短按钮
- 富预览卡片展示每个链接的缩略图、标题和描述
- 仪表盘包含可排序、可搜索的表格
- 响应式布局，适合在移动端快速创建链接

## 技术说明
- 可使用任意技术栈和库
- 推荐免费 API：jsonlink.io、microlink.io，或服务端抓取 + HTML 解析
- 优先实现功能，再进行优化打磨
