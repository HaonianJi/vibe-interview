# 短链接服务

构建一个短链接应用，用户可以缩短长 URL、选择性设置自定义别名、查看带图表的点击分析，并通过仪表盘管理所有链接。

## 需求

### 核心功能（必须实现）
1. **URL 缩短**：接受长 URL 并生成唯一的短码——展示生成的短链接并提供一键复制按钮
2. **自定义别名**：允许用户指定自定义别名替代随机短码，校验防止重复
3. **跳转处理**：访问短链接时重定向到原始长 URL，并记录点击事件的时间戳和基本元数据
4. **点击分析**：为每个短链接展示总点击次数、过去 7 天点击量的时序图表，以及来源分布（如可获取）
5. **链接管理仪表盘**：表格列出所有已创建的短链接，包含短 URL、原始 URL、创建日期和点击次数等列，支持搜索和排序

### 界面要求
- 简洁的落地页，突出显示 URL 输入框和缩短按钮
- 仪表盘视图包含可排序、可搜索的链接表格
- 每个链接的分析详情页，至少包含一个可视化点击数据随时间变化的图表
- 响应式布局，适合在移动端快速创建链接

## 技术说明
- 可使用任意技术栈
- 可使用任意库
- 优先实现功能，再进行优化打磨
