# 看板

构建一个用于项目管理的交互式看板。

## 需求

### 核心功能（必须实现）
1. **列**：至少包含 3 个默认列——"待办"、"进行中"、"已完成"
2. **任务卡片**：创建包含标题、描述、优先级（低/中/高）和可选截止日期的任务
3. **拖拽操作**：通过拖拽在列之间移动任务
4. **编辑与删除**：支持内联编辑或通过弹窗编辑任务详情；删除时需确认
5. **数据持久化**：所有看板状态在页面刷新后必须保留

### 加分功能（可选）
- 按标题或优先级搜索/筛选任务
- 列管理（添加/重命名/删除列）
- 每列显示任务数量

### 界面要求
- 优先级可视化标识（颜色编码）
- 流畅的拖拽交互体验
- 响应式布局

## 技术说明
- 可使用任意技术栈
- 拖拽功能可使用任意库（dnd-kit、react-beautiful-dnd 等）或原生 HTML5 拖拽 API
- 注重流畅、直观的用户体验
