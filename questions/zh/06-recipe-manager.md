# 食谱管理器

构建一个食谱管理应用，用户可以创建、浏览和整理食谱，按食材搜索，并从选定的食谱生成购物清单。

## 需求

### 核心功能（必须实现）
1. **食谱增删改查**：创建、查看、更新和删除食谱，包含标题、描述、食材列表（含用量和单位）以及有序的制作步骤
2. **按食材搜索**：按食材名称搜索和筛选食谱——展示包含一种或多种所查询食材的食谱
3. **购物清单生成**：选择一个或多个食谱，生成合并后的购物清单，自动合并重复食材并累加用量
4. **食谱详情页**：以清晰的布局展示单个食谱，食材以清单形式呈现，步骤以编号列表呈现
5. **食谱分类**：为食谱指定分类或标签（如早餐、晚餐、素食等），支持按分类筛选食谱列表

### 界面要求
- 主页面以列表/网格视图展示食谱，配有搜索栏和分类筛选控件
- 食谱创建/编辑表单支持动态添加/移除食材和步骤
- 购物清单面板或页面支持勾选已购项目
- 响应式布局，适配桌面端和移动端

## 技术说明
- 可使用任意技术栈
- 可使用任意库
- 优先实现功能，再进行优化打磨
