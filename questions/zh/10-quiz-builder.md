# 测验生成器

构建一个测验应用，能够从免费 API 导入真实知识问答题目，也支持自定义创建测验，并提供完整的评分和答案回顾功能。

## 需求

### 核心功能（必须实现）
1. **从 API 导入**：从 **Open Trivia Database API** 获取题目——用户选择分类、难度和题目数量，导入到新测验中
2. **自定义创建测验**：同时支持手动创建测验，包含标题、描述和自定义题目——支持选择题、判断题和简答题三种题型
3. **答题模式**：逐题展示，配有相应的输入控件；显示进度，并允许在题目间自由导航
4. **评分与结果**：提交后计算并展示总分、百分比以及每道题的结果
5. **答案回顾**：回顾页面展示每道题目、用户的答案和正确答案，并以视觉标识区分对错

### API 集成
- 使用 **Open Trivia Database API**（opentdb.com/api.php）获取真实知识问答题目
- 让用户浏览 API 提供的可用分类（动态获取分类列表）
- 支持按难度（easy/medium/hard）和题型（选择题/判断题）筛选
- 正确解码 API 响应中的 HTML 实体
- 优雅处理 API 错误和空响应

### 界面要求
- 测验列表页展示所有测验（导入的和自定义的），包含标题、题目数量和来源标识
- 导入向导：选择分类 → 难度 → 数量 → 预览 → 确认
- 逐步式答题界面，带有清晰的进度指示
- 测验编辑器的表单布局简洁

## 技术说明
- 可使用任意技术栈和库
- Open Trivia DB 文档：https://opentdb.com/api_config.php
- 优先实现功能，再进行优化打磨
