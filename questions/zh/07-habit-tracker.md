# 习惯追踪器

构建一个习惯追踪应用，用户可以定义每日习惯、记录完成情况，通过日历热力图可视化坚持度，并查看每周统计数据。

## 需求

### 核心功能（必须实现）
1. **习惯管理**：创建、编辑和删除习惯，包含名称、可选描述和目标频率（如每天、仅工作日）
2. **每日打卡**：今日视图列出所有活跃习惯，提供开关按钮标记当天的完成或未完成状态
3. **连续打卡统计**：自动计算并展示每个习惯的当前连续天数和最长连续天数
4. **日历热力图**：展示类似 GitHub 风格的热力图网格，覆盖过去 12 周，颜色深浅表示每天完成的习惯数量
5. **每周统计**：统计面板展示本周和上周每个习惯的完成率，以柱状图或进度条呈现

### 界面要求
- 仪表盘首页展示今日习惯和整体连续打卡摘要
- 热力图使用颜色编码并附带图例说明
- 习惯完成标记的切换交互要流畅
- 响应式设计，适合日常移动端使用

## 技术说明
- 可使用任意技术栈
- 可使用任意库
- 优先实现功能，再进行优化打磨
