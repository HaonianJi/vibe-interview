# File Explorer

Build a virtual file explorer application with a tree-view sidebar, breadcrumb navigation, and the ability to create, rename, delete, and preview files and folders.

## Requirements

### Core Features (Must Have)
1. **Tree View**: A collapsible tree sidebar showing the full folder hierarchy with expand/collapse toggles and file/folder icons
2. **File & Folder Operations**: Create new files and folders, rename existing items, and delete items with a confirmation prompt
3. **Breadcrumb Navigation**: A breadcrumb bar at the top reflecting the current path, with each segment clickable to navigate up
4. **Content Preview**: Clicking a file shows a preview panel — render plain text content, and show placeholder previews for image or other types
5. **Drag and Drop**: Move files and folders by dragging them to a different folder in the tree or in the main content area

### UI Requirements
- A two-panel layout with the tree sidebar on the left and the content/preview area on the right
- Visual feedback on drag-and-drop hover targets
- Context menus or action buttons for create, rename, and delete operations
- Responsive behavior that collapses the sidebar on smaller screens

## Technical Notes
- You may use any tech stack
- You may use any libraries
- Focus on functionality first, then polish
