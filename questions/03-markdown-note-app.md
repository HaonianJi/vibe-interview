# Markdown Note-Taking App

Build a note-taking application with Markdown support.

## Requirements

### Core Features (Must Have)
1. **Note CRUD**: Create, read, update, and delete notes
2. **Markdown Editor**: Text area with Markdown syntax support
3. **Live Preview**: Side-by-side or toggle view showing rendered Markdown
4. **Note List**: Sidebar listing all notes, sorted by last modified
5. **Search**: Search notes by title and content
6. **Data Persistence**: All notes must persist across page refreshes

### Markdown Support
Must render at least: headings, bold, italic, lists, code blocks, links

### UI Requirements
- Split-pane layout (note list | editor | preview) or equivalent
- Currently selected note highlighted in sidebar
- Auto-save or explicit save button

## Technical Notes
- You may use any tech stack
- You may use a Markdown rendering library (marked, react-markdown, etc.)
- Focus on a smooth editing experience
