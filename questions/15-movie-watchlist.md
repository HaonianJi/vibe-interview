# Movie Watchlist

Build a movie watchlist application where users can search for movies using a free API, add them to a personal watchlist, write ratings and reviews, and filter by genre or rating.

## Requirements

### Core Features (Must Have)
1. **Movie Search**: Search for movies by title using a free API (e.g., OMDB, TMDB) and display results with poster, title, year, and genre
2. **Watchlist Management**: Add movies from search results to a personal watchlist, mark movies as watched or unwatched, and remove movies from the list
3. **Rate & Review**: For any movie on the watchlist, assign a star rating (1-5) and write a short text review — display both on the movie detail view
4. **Filter & Sort**: Filter the watchlist by genre, watched/unwatched status, or minimum rating, and sort by title, year, or user rating
5. **Movie Detail View**: A detail page showing full movie information from the API (plot, director, cast, runtime) alongside the user's rating and review

### UI Requirements
- A search bar with auto-triggered or button-triggered results displayed as a grid or list of movie cards
- A watchlist page with filter controls in a sidebar or toolbar
- Star rating input component for rating movies
- Responsive card layout for both search results and watchlist views

## Technical Notes
- You may use any tech stack
- You may use any libraries
- You may use any free movie API (OMDB, TMDB, etc.) — include a note about obtaining an API key
- Focus on functionality first, then polish
