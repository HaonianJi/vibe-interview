# Personal Finance Tracker

Build a web application that helps users track their personal finances.

## Requirements

### Core Features (Must Have)
1. **Transaction Management**: Add, edit, and delete income/expense transactions with amount, category, date, and description
2. **Dashboard**: Show current balance, total income, total expenses for the current month
3. **Category Breakdown**: Visual chart (pie or bar) showing spending by category
4. **Transaction List**: Sortable and filterable list of all transactions
5. **Data Persistence**: All data must persist across page refreshes

### Categories
Support at least these categories: Food, Transport, Housing, Entertainment, Shopping, Health, Education, Other

### UI Requirements
- Clean, responsive design that works on desktop
- Form validation (no negative amounts, required fields)
- Confirmation before deleting transactions

## Technical Notes
- You may use any tech stack
- You may use any UI library or CSS framework
- Focus on functionality first, then polish
