# Recipe Manager

Build a recipe management application where users can create, browse, and organize their recipes, search by ingredient, and generate shopping lists from selected recipes.

## Requirements

### Core Features (Must Have)
1. **Recipe CRUD**: Create, read, update, and delete recipes with a title, description, ingredient list (with quantities and units), and ordered preparation steps
2. **Ingredient-Based Search**: Search and filter recipes by ingredient name — show recipes that contain one or more queried ingredients
3. **Shopping List Generator**: Select one or more recipes and generate a consolidated shopping list that merges duplicate ingredients and sums quantities
4. **Recipe Detail View**: Display a single recipe with a clean layout showing ingredients as a checklist and steps as a numbered list
5. **Recipe Categories**: Assign categories or tags (e.g., breakfast, dinner, vegan) and filter the recipe list by category

### UI Requirements
- A main recipe list/grid view with search bar and category filter controls
- A recipe creation/edit form with dynamic fields for adding/removing ingredients and steps
- A shopping list panel or page that allows checking off items
- Responsive layout that works on both desktop and mobile widths

## Technical Notes
- You may use any tech stack
- You may use any libraries
- Focus on functionality first, then polish
