# Quiz Builder

Build a quiz application where users can create quizzes with various question types, take those quizzes, receive a score, and review correct and incorrect answers.

## Requirements

### Core Features (Must Have)
1. **Quiz Creation**: Create a quiz with a title, description, and an ordered list of questions — support at least multiple-choice, true/false, and short-answer question types
2. **Question Editor**: A dynamic form for adding, removing, and reordering questions within a quiz, with type-specific fields (e.g., options for multiple-choice, correct answer for all types)
3. **Quiz Taking**: A quiz-taker view that presents one question at a time (or all at once) with appropriate input controls for each question type
4. **Scoring & Results**: After submission, calculate and display the total score, percentage, and per-question result (correct/incorrect)
5. **Answer Review**: A review screen showing each question with the user's answer alongside the correct answer, with visual indicators for right and wrong

### UI Requirements
- A quiz list page showing all available quizzes with title and question count
- A step-by-step or scrollable quiz-taking interface with clear progress indication
- A results summary page with score breakdown and a link to detailed review
- Clean form layout for the quiz editor with drag-to-reorder or arrow buttons for question order

## Technical Notes
- You may use any tech stack
- You may use any libraries
- Focus on functionality first, then polish
